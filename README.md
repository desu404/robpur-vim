## A vim colorscheme made with VimArtisan by [@mserdarsanli](https://github.com/mserdarsanli)

# Using vim-plug
```
Plug `skurob/robpur-vim`
```

or 

# Using source
```
source `file/path/robpur.vim`
```

# Applying
```
set background=dark
colorscheme robpur
```
